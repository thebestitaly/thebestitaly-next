🔧 Implementation Guide

1. Include the Widget Script

Add this script tag to your HTML page:

<script src="https://thebestitaly.eu/widgets/widget.js"></script>

2. Add Widget HTML

Create a container with the required data attributes:

<!-- Destination Widget -->
<div id="my-widget" 
     data-tbi-widget
     data-type="destination" 
     data-id="roma" 
     data-size="medium" 
     data-theme="light" 
     data-language="en"
     data-show-selector="true">
</div>

<!-- Company Widget -->
<div id="company-widget" 
     data-tbi-widget
     data-type="company" 
     data-id="osteria-del-borgo" 
     data-size="small" 
     data-theme="dark" 
     data-language="it"
     data-show-selector="true">
</div>

<!-- Article Widget -->
<div id="article-widget" 
     data-tbi-widget
     data-type="article" 
     data-id="best-restaurants-tuscany" 
     data-size="large" 
     data-theme="auto" 
     data-language="fr"
     data-show-selector="true">
</div>

3. Automatic Initialization

Widgets initialize automatically when the page loads. No additional JavaScript required!

📝 Important Notes
  • Company URLs: All companies use the format /poi/slug regardless of the company ID
  • Language Dropdown: Always visible by default - no "Discover more" button, just direct language selection
  • Content Integration: Large widgets load full page content directly from the database
  • Real-time Data: All content is fetched live from TheBestItaly's API

⚙️ Widget Parameters

data-tbi-widget
Required identifier for widget initialization
Required (no value needed)
data-type
Type of content to display
destination | company | article
data-id
Unique identifier or slug for the content
String (e.g., "roma", "osteria-del-borgo")
data-size
Widget size and layout
small | medium | large
data-theme
Visual theme for the widget
light | dark | auto
data-language
Initial language (see supported languages below)
Language code (e.g., "en", "it", "fr")
data-show-selector
Show/hide language dropdown selector
true | false (default: true)
